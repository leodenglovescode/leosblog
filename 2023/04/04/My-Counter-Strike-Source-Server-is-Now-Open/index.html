My Counter-Strike: Source Server is Now Open!

Leodeng
2023-04-04

Long time no see!


I’ve been having a lot a trouble on schoolwork and time recently, so no time for blogs, during the Qing Ming Festival Break in China (Similiar to Mexico’s Day of the Dead but not quite) ,I finally can have a little break where I can make projects I like!

As I said before, I am obsessed with old games and old tech, which led me to think about old Online Multiplayer Games such as GMOD, Counter Strike 1.6, Source and more, so I thought: Why not make a old Game server to bring that sense of nostalgia back? Well I did it, looking for tutorials and asking on the AlliedModders Forum for a lot of time, and setting up all kinds of plugins. Now what I Have is a Full On DeathMatch Server running on my old Thinkpad with Upgraded 8 GB RAM and 128GB SSD, which working amazingly good in 2023!



I Will Post The IP Address of my server Later in the Month, and I wish you all a good holiday!
